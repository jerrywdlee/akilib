How to use 使い方
-----------------------------------------------
sudo apt-get install i2c-tools python-smbus

::

    from akilib import AKI_I2C_HDC1000

    x = AKI_I2C_HDC1000(1)

    x.Config()

    x.Temperature()

    x.Humidity()
